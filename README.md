# Njupt Survival Guide
南京邮电大学生存指南，用于导航和整合各种公开的资源和信息，减少大家的信息差


## 快速开始开发
```bash
git clone https://github.com/NJUPT-NAVI/NJUPT-Survival-Guide.git
cd NJUPT-Survival-Guide
npm install
npm run dev
```
